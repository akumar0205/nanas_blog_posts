# Nana's Blog Posts Backup

This repository is a personal backup of my grandpa's blog posts and poetry.

It preserves his original writing (mostly in Hindi, with some English posts) in a simple static website so family and friends can continue to read and share his work.

## What this repo contains

- `posts/`: Markdown source files for each post.
- `public/`: Static site files (`index.html`, `style.css`, `script.js`, `posts.json`) used to render the archive.

## Why this exists

Blog platforms can change, shut down, or remove old content. This backup keeps my grandpa's words safe and accessible in one place.
